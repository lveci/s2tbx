OTB Installation

OTB Standalone Tool Installation

Before integrating it in SNAP, OTB needs to be deployed. Two options are available:
  • Portable version
  • Installable version: installer to execute
OTB can be freely downloaded from its website:
Download and install/unpack OTB before integrating it in SNAP.

OTB Integration in SNAP

Once OTB is unpacked or installed you have to install the plugin.
First define the OTB_BIN_DIR environment variable as the full path to the "bin" directory of your OTB installation. Then from SNAP, go to Tools/Plugins, under the "Available Plugins" tab the OTB appears (if it doesn't, check your internet connection and your proxy settings from tools/options>WWW). Tick the checkbox next to it, click install and follow the instructions.
